# 头条项目开发流程

## 一、总览

开发一个新模块，按标准四步走：

```
Step 1: models/xxx.py     定义数据库表（SQLAlchemy ORM）
Step 2: schemas/xxx.py     定义 Pydantic 模型（控制接口字段）
Step 3: crud/xxx.py        写数据库操作函数
Step 4: routers/xxx.py     写路由，调 crud + schemas + success_response
```

**口诀**：`模型 → 模式 → CRUD → 路由`

---

## 二、Step 1：models — 定义 ORM 模型

```python
# models/news.py

from sqlalchemy import String, Text, DateTime, ForeignKey, Index
from sqlalchemy.orm import Mapped, mapped_column
from models.base import Base


class News(Base):
    __tablename__ = "news"

    id: Mapped[int] = mapped_column(primary_key=True, autoincrement=True)
    title: Mapped[str] = mapped_column(String(255), nullable=False)
    content: Mapped[str] = mapped_column(Text, nullable=False)
    category_id: Mapped[int] = mapped_column(ForeignKey("news_category.id"))
```

**规则：**
- 继承 `models.base.Base`
- 用 SQLAlchemy 2.0 风格：`Mapped[T]` + `mapped_column()`
- 只负责**表结构映射**，不做数据转换

---

## 三、Step 2：schemas — 定义 Pydantic 模型

```python
# schemas/news.py

from datetime import datetime
from typing import Optional
from pydantic import BaseModel, Field, ConfigDict


class NewsItemResponse(BaseModel):
    id: int
    title: str
    description: Optional[str] = None
    image: Optional[str] = None
    publish_time: datetime = Field(..., alias="publishTime")
    category_id: int = Field(..., alias="categoryId")
    views: int

    model_config = ConfigDict(from_attributes=True)
```

**关键点：**

| 配置/方法 | 作用 |
|---|---|
| `from_attributes=True` | 允许从 ORM 对象用 `点号` 读属性 |
| `model_validate(obj)` | 把 ORM 对象转成 Pydantic 模型 |
| `Field(..., alias="camelCase")` | Python 写 snake_case，JSON 输出 camelCase |
| `Field([], alias="xxx")` | 设置默认值 |

---

## 四、Step 3：crud — 写数据库操作

```python
# crud/news.py

from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession
from models.news import News


async def get_news_list(db: AsyncSession, category_id: int, skip: int = 0, limit: int = 100):
    stmt = select(News).where(News.category_id == category_id).offset(skip).limit(limit)
    result = await db.execute(stmt)
    return result.scalars().all()
```

**规则：**
- 只返回 **ORM 对象**，不做 dict 转换
- 返回 ORM 对象，让 schema 控制哪些字段出去
- 纯 SELECT 不需要 `commit`
- INSERT / UPDATE / DELETE 需要 `await db.commit()`

## 五、Step 4：routers — 写路由

```python
# routers/news.py

from fastapi import APIRouter, Depends
from sqlalchemy.ext.asyncio import AsyncSession
from config.db_conf import get_db
from utils.response import success_response
from crud.news import get_news_list as crud_get_news_list, get_news_count as crud_get_news_count
from schemas.news import NewsItemResponse

router = APIRouter(prefix="/api/news", tags=["news"])


@router.get("/list")
async def get_news_list(
    db: AsyncSession = Depends(get_db),
    category_id: int = Query(..., alias="categoryId"),
    page: int = 1,
    page_size: int = 100,
):
    offset = (page - 1) * page_size
    news_list = await crud_get_news_list(db, category_id, skip=offset, limit=page_size)
    total = await crud_get_news_count(db, category_id)
    has_more = (offset + len(news_list)) < total

    return success_response(data={
        "list": [NewsItemResponse.model_validate(n) for n in news_list],
        "total": total,
        "hasMore": has_more,
    })
```

**路由里的三步：**
1. 调 **crud** 拿数据（ORM 对象）
2. 用 **schema.model_validate()** 转成 Pydantic
3. 用 **success_response()** 统一包一层返回

## 七、开发注意事项

### 1. 不要手动拼 dict

```python
# ❌ 错误：手动拼 dict
return [{
    "id": n.id,
    "title": n.title,
    "publishTime": n.publish_time,
} for n in related]

# ✅ 正确：返回 ORM，用 schema 转换
return result.scalars().all()
```

### 2. 不要直接返回 ORM 对象

```python
# ❌ 错误：ORM 对象直接丢给 success_response
return success_response(data=news_list)   # jsonable_encoder 虽然会处理，但字段不可控

# ✅ 正确：先转 schema，再返回
data = [NewsItemResponse.model_validate(n) for n in news_list]
return success_response(data=data)
```

### 3. schema 控制字段白名单

只在 Pydantic 模型里写**想暴露的字段**，没写的字段自动过滤。

```python
class UserOut(BaseModel):
    id: int
    username: str
    # 故意不写 password，密码不会返回给客户端

    model_config = ConfigDict(from_attributes=True)
```

### 4. camelCase 驼峰命名

```python
publish_time: datetime = Field(..., alias="publishTime")
category_id: int = Field(..., alias="categoryId")
```

Python 里用 `obj.publish_time`，JSON 里变成 `publishTime`。
