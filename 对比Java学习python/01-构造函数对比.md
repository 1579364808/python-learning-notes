# 构造函数对比：Java vs Python

## Java 写法（通过重载多个构造函数）

```java
public class Client {
    private String apiKey;
    private String baseUrl;
    private int timeout;

    // 全参构造函数
    public Client(String apiKey, String baseUrl, int timeout) {
        this.apiKey = apiKey;
        this.baseUrl = baseUrl;
        this.timeout = timeout;
    }

    // 部分参数构造函数（重载）
    public Client(String apiKey, String baseUrl) {
        this(apiKey, baseUrl, 30); // 默认 timeout=30
    }

    // 更少参数的构造函数
    public Client(String apiKey) {
        this(apiKey, "https://default.url", 30);
    }
}
```

## Python 写法（关键字参数 + 默认值，一个构造函数搞定）

```python
class Client:
    def __init__(self, api_key=None, base_url=None, timeout=None):
        self.api_key = api_key
        self.base_url = base_url
        self.timeout = timeout

# 调用时可以指定任意参数组合
client = Client(api_key=self.apiKey, base_url=self.baseUrl, timeout=timeout)
```

## 核心区别

| 对比项    | Java       | Python              |
| ------ | ---------- | ------------------- |
| 构造函数数量 | 多个（方法重载）   | 只有一个（`__init__`）    |
| 默认参数   | 通过重载实现     | 参数默认值直接写在定义中        |
| 参数传递   | 按位置匹配      | 支持关键字参数 `key=value` |
| 灵活性    | 需要提前定义所有组合 | 运行时灵活传参             |

## 一句话总结

Python 的 `__init__` 就是 Java 的构造函数。Java 需要写多个重载版本，Python 靠关键字参数 + 默认值一个方法搞定，调用时 `key=value` 明确指定参数，更简洁灵活。
