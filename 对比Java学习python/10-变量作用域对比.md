## 变量作用域对比

### 一句话定性
Python **只有函数级作用域**（无块级作用域），for/if 内部定义的变量外部仍可访问；Java **有块级作用域**，花括号内定义的变量外部不可见。

### 代码对比

```python
# Python：for 块不构成独立作用域
for i in range(3):
    x = i * 2
print(x)  # 输出 4，x 在循环外仍然存在！

# 空循环时的问题
items = []
for item in items:
    msg = item
print(msg)  # NameError: name 'msg' is not defined

# if 同理
if False:
    y = 10
print(y)  # NameError
```

```java
// Java：块级作用域
String x = null;  // 必须在外层声明
for (int i = 0; i < 3; i++) {
    x = String.valueOf(i * 2);  // 复用外层变量
}
System.out.println(x);  // OK

// int i 定义在 for 里，外部不可见
System.out.println(i);  // ❌ 编译报错
```

### 差异表格

| 对比项 | Python | Java |
|--------|--------|------|
| 作用域粒度 | **函数级** | **块级** |
| for/if 内定义 | 外部可访问 | 外部不可访问 |
| 空循环后果 | `NameError`（变量未定义） | 编译错误（变量未初始化）|
| 防御写法 | 在外层先赋值默认值 | 在外层先声明并初始化 |

### 要点说明

1. **Python 没有块级作用域：** `for`、`if`、`while` 内部定义的变量，外部都可以访问
2. **踩坑点：** 如果循环或条件从未执行，变量就从未定义，访问会抛 `NameError`
3. **规范写法：** 在外层先给变量一个兜底值（如 `result = ""` 或 `result = None`）
4. **Java 相反：** 块内变量块外不可见，所以必须在块外先声明才能跨块使用
